Project#2 Warrior's Tale

Introduction:

Warrior's Tale is an innovative 2D side-scrolling action game developed utilizing the Unity engine.

Project#2 Warrior's Tale

Input System:

This game integrates Unity's latest Input System. This system allows for real-time swapping between operation modes on various devices and controllers. Players can smoothly transition from using a mouse-and-keyboard setup to an Xbox gamepad, thereby elevating the gameplay experience and catering to individual preferences in terms of controls.

Animator:

Warrior's Tale uses animation technologies, including Blend Trees and State Machines. These technologies help render a rich array of animations and facilitate dynamic transitions between different animations. These mechanisms offer players a visually compelling experience, with characters that move naturally and respond immediately to player input.

Project#2 Warrior's Tale

Rule Tile & Animated Tile:

The integration of Rule Tiles and Animated Tiles has revolutionized the way we design game levels. These tools allow developers to create and customize tile templates, significantly simplifying the map-creation process. Moreover, Animated Tiles provide the maps with a degree of dynamism, adding life to the environment and enhancing the immersive qualities of the game world.

Scriptable Object:

The use of Scriptable Objects has been integral to the game's development. These objects serve to store critical game data and manage game events through broadcasting and listening systems. By utilizing Scriptable Objects, we have effectively reduced the coupling between different segments of our code. This results in a more independent, modular collaboration between game scripts, contributing to a well-structured and maintainable codebase.

Self Reflection

This project is my attempt at a Unity 2D platformer action game. In this project I mainly made it with the purpose of learning. After completing this project, I have improved my understanding of the Unity game production process, especially for 2D games. I learnt and mastered the advanced use of Tile Map, Animator and the latest Input System.
